# OSDR
Open Standard Digital Radio (OSDR) is a clean sheet approach to audio broadcasting, exploring the potential of modern codecs like OAC and AVIF.
Using only 100 kHz bandwidth on the VHF-II band and OFDM it can transmit a single stereo radio program in CD quality with RDS like service information while being fully SFN capable.

The first version of the draft is currently under development.
